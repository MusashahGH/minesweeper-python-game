# 💣 Minesweeper Game (Python)

A classic Minesweeper game built for the console using core Python concepts — 2D lists, nested loops, and functions.

## 🎮 How It Works

- The game creates a 5x5 grid and randomly places a user-defined number of mines (1 to 25) on the board.
- The player guesses a row and column to reveal a cell.
- If the cell has a mine, the game ends and shows the solution.
- If the cell is safe, it displays the number of mines surrounding that cell (like traditional Minesweeper).
- The player wins by revealing all safe cells without hitting a mine.

## 🧠 Concepts Used
- 2D Lists (Arrays) for board representation
- Nested `while` and `for` loops
- Functions (`checkMineAround`, `displaySol`, `displayBoard`)
- Random mine placement using the `random` module
- Input validation

## 🚀 How to Run
1. Clone or download this repository
2. Make sure Python 3 is installed
3. Run the file:
```bash
   minesweeper_game_python.py
```
4. Enter the number of mines you want (1–25)
5. Start guessing rows and columns to play!
